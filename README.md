# 1986-corvette-telematics-dashboard

## Overview
This project is a Raspberry Pi-based telematics dashboard designed for a pre-OBD 1986 Chevrolet Corvette. The system reads real-time engine data from the vehicle's 160-baud ALDL diagnostic port, decodes key sensor values, and displays them through a live web dashboard that can be accessed from a phone or laptop.

## My Contribution
My main responsibility was developing the web interface for the system. I used Flask, HTML, CSS, and JavaScript to display decoded vehicle data in a clear and readable format. I also worked on live dashboard updates, secure login access, trip logs, summary statistics, and derived vehicle-health indicators.

## Technologies Used
- Python
- Flask
- HTML/CSS/JavaScript
- Raspberry Pi

## Features
- Displays live vehicle telemetry on a web dashboard
- Shows sensor values such as RPM, coolant temperature, throttle position, battery voltage, and vehicle speed
- Provides secure login access
- Supports trip logging and trip summaries
- Calculates derived vehicle-health metrics
- Allows dashboard access from phones or laptops on the same network
